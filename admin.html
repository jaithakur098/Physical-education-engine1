CSV / Excel Header Format

Ensure your file has these EXACT column headers:

  • Question - The text description
  • Option A, Option B, Option C, Option D
  • Answer - The correct choice index (0=A, 1=B, 2=C, 3=D)
  • Explanation - (Optional) Explanation text
  • Image - (Optional) Illustration link / Base64

JSON Array Structure

Import valid JSON arrays using this format:

[
  {
    "question": "The question description text goes here...",
    "options": ["Choice A", "Choice B", "Choice C", "Choice D"],
    "answer": 0,
    "explanation": "Optional feedback description text..."
  }
]
